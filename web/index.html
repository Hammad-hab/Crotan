v0.1 Croton Beta

Introducing Croton,

A futuristic, Python powered programming language for writing multilayered data cleaning systems

Get Croton source using: 
git clone https://github.com/Hammad-hab/Croton.git

Made With Transparency

Croton has been designed to allow programmers to customize the behavior of the language constructs like If-conditions, Loops, variable declarations and more!

Nothing Crazy Underneath

Data cleaning is complicated as it is, Croton aims to change that not by adding more complexity to the process but by introducing a hierarchical system which makes it easy to understand what's going on!

Simple as Simple can be

Even though Croton is not as Readable as Python, it does have a syntax that is readable and an interpreter that's easy to understand and customize.

Hierarchical Pattern

The best way to clean data is to do it in a hierchical pattern. It is much more easier to clean data if you're working in a step-by-step component-wise enviornment where data flow can be controlled in an effective way.
